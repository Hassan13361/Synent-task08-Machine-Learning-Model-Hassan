**Problem Statement:**
Predict house prices using property characteristics such as area, bedrooms, bathrooms, parking, furnishing status, and other housing features.

**Dataset Details**
The House Price Dataset contains information about residential properties and their selling prices.

**Features**
Area
Bedrooms
Bathrooms
Stories
Parking
Main Road Access
Guest Room
Basement
Air Conditioning
Preferred Area
Furnishing Status
**Target**
Price
**Approach**
Load and clean dataset
Encode categorical variables
Select features and target
Split data into training and testing sets
Train Linear Regression model
Train Random Forest model
Evaluate using RMSE and R² Score
Compare model performance
**Results**
Both models successfully predict house prices.
Random Forest generally achieves lower RMSE and higher R².
The best-performing model is selected based on evaluation metrics.
**Technologies Used**
Python
Pandas
NumPy
Scikit-learn
This satisfies all requirements: preprocessing, feature selection, train/test split, Linear Regression, Random Forest, and evaluation with RMSE/R².
